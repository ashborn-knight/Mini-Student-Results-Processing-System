# Mini Student Results Processing System

A C# console application that processes student results for 10 students across 5 courses.

Built as part of CE 380 – Programming with C# at the University of Mines and Technology, Tarkwa.


What the Program Does


Allows the user to enter details and scores for 10 students
Calculates total score, average score, grade and academic status for each student
Displays a full results report for all students
Validates that all scores entered are between 0 and 100
