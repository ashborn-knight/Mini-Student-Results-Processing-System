# 📝 Mini Student Results Processing System

A C# Console Application designed to process and manage student academic results.  
This project demonstrates practical programming concepts and structured application design.

---

## 📖 Project Overview

The **Mini Student Results Processing System** provides functionality to:

- Capture student details (Name, ID, Programme, Level)
- Record scores for multiple courses
- Automatically calculate total and average scores
- Assign grades based on performance
- Determine academic status (Pass/Fail)
- Generate and display detailed student reports

---

## 🚀 Features

- Menu-driven interface for easy navigation
- Input validation to ensure scores are between 0 and 100
- Automatic grade assignment based on average score
- Academic status determination (Passed/Failed)
- Report generation with all student details and computed results
